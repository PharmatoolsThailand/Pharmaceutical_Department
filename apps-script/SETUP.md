# Pharmacy Hub — ตั้งค่า registry บน Google Sheet

รายการแอพทั้งหมดเก็บใน Google Sheet → **เพิ่มแอพ = เพิ่ม 1 แถว** ไม่ต้องแก้โค้ด

## 1. สร้างชีท
1. สร้าง Google Sheet ใหม่ → ตั้งชื่อชีท (แท็บล่าง) ว่า **`Apps`**
2. แถวที่ 1 ใส่หัวคอลัมน์ตามนี้ (ต้องสะกดตรง):

| id | name | desc | category | icon | path | tags | status | public | order |
|----|------|------|----------|------|------|------|--------|--------|-------|

ความหมายคอลัมน์:
- **id** — รหัสไม่ซ้ำ (เช่น `had`, `renaldose`)
- **name** — ชื่อแอพที่แสดงบนการ์ด
- **desc** — คำอธิบายสั้น (1–2 บรรทัด)
- **category** — `dosing` / `forms` / `admin` / `tools` / `other` (เพิ่มหมวดใหม่ได้ใน `js/config.js`)
- **icon** — อิโมจิ เช่น `💊`
- **path** — ลิงก์แอพ: URL เต็ม (`https://…`) หรือ path ภายใน (`../HADCalculator/index.html`)
- **tags** — คั่นด้วยจุลภาค `dose, iv, infusion` (ใช้ตอนค้นหา)
- **status** — `active` / `beta` / `soon` (soon = แสดงแต่กดไม่ได้)
- **public** — `TRUE` = โชว์สาธารณะ (ไม่ต้อง login เห็น) · `FALSE`/ว่าง = เฉพาะหลัง login (แอพเว้นว่าง = สาธารณะ)
- **order** — ลำดับในหมวด (เลขน้อยขึ้นก่อน)

> ค่าตั้งต้น 6 แอพดูได้ใน `js/config.js` (`APPS_SEED`) คัดลอกมาวางเป็นแถวแรก ๆ ได้เลย

### ชีท Docs (เอกสารสำคัญ — โชว์เฉพาะหลัง login)
สร้างแท็บที่สองชื่อ **`Docs`** แถวที่ 1 ใส่หัวคอลัมน์:

| id | title | url | note | public | order |
|----|-------|-----|------|--------|-------|

- **id** — รหัสไม่ซ้ำ (เพิ่มจากหน้าเว็บจะสร้างให้อัตโนมัติ `doc-...`)
- **title** — ชื่อเอกสาร · **url** — ลิงก์ · **note** — หมายเหตุสั้น
- **public** — `TRUE` = โชว์สาธารณะ · `FALSE`/ว่าง = เฉพาะหลัง login (เอกสารเว้นว่าง = เฉพาะเจ้าหน้าที่)
- **order** — ลำดับ

> ถ้าไม่มีแท็บ `Docs` สคริปต์จะสร้างให้อัตโนมัติเมื่อบันทึกลิงก์แรกจากหน้าเว็บ

### ชีท Categories (หมวดหมู่แอป — Admin เพิ่ม/แก้/ลบได้)
สร้างแท็บชื่อ **`Categories`** แถวที่ 1 ใส่หัวคอลัมน์:

| id | name | icon | order |
|----|------|------|-------|

- **id** — รหัสไม่ซ้ำ (เพิ่มจากหน้าเว็บสร้างให้อัตโนมัติ `cat-...`) · ค่าในคอลัมน์ `category` ของชีท Apps ต้องตรงกับ id นี้
- **name** — ชื่อหมวด · **icon** — อิโมจิ · **order** — ลำดับ

> ถ้าแท็บ `Categories` ว่าง/ไม่มี ระบบใช้หมวดตั้งต้น 5 หมวดใน `js/config.js` (`CATS_SEED`) · เพิ่มแถวเองในชีทหรือผ่านเมนู จัดการ ▸ หมวดหมู่แอป

## 2. ติดตั้ง Apps Script
1. ในชีท: **Extensions ▸ Apps Script**
2. ลบโค้ดเดิม วางเนื้อหา `Code.gs` (ในโฟลเดอร์นี้) ลงไป → Save
3. **Deploy ▸ New deployment ▸** เลือกชนิด **Web app**
   - Execute as: **Me**
   - Who has access: **Anyone**
4. คัดลอก **Web app URL** (ลงท้าย `/exec`)

## 3. ตั้งรหัสผ่าน 2 ระดับ (สิทธิ์ผู้ใช้ 3 ระดับ)
| ระดับ | login | สิทธิ์ |
|------|-------|-------|
| ทั่วไป (guest) | ไม่ต้อง | เห็นเฉพาะแอพ/เอกสารที่ตั้ง `public` |
| เจ้าหน้าที่ (staff) | รหัส `STAFF_TOKEN` | เห็น/ใช้แอพ+เอกสารที่ซ่อนได้ (แก้ไม่ได้) |
| ผู้ดูแล (admin) | รหัส `ADMIN_TOKEN` | จัดการ/ตั้งค่า/สลับการโชว์ ได้ทุกอย่าง |

ตั้งรหัสฝั่งเซิร์ฟเวอร์ (เก็บใน Apps Script):
- **Project Settings (⚙) ▸ Script Properties ▸ Add** → ใส่ 2 key: `STAFF_TOKEN` และ `ADMIN_TOKEN`
- หรือเปิด `Code.gs` แก้รหัสในฟังก์ชัน `setup_setTokens()` แล้วรัน 1 ครั้ง

> การเขียนชีท (เพิ่ม/แก้/ลบ/สลับ public) ต้องเป็น **admin** เท่านั้น · ไม่ตั้งรหัส = ปฏิเสธทุกการเขียน · การอ่าน/แสดงทำได้ปกติ
> โหมด local (ยังไม่ตั้ง `APPS_SCRIPT_URL`) ใช้รหัสทดสอบใน `js/config.js` (`LOCAL_STAFF_PW`/`LOCAL_ADMIN_PW` = `staff`/`admin`)

## 4. เชื่อมกับ Hub
เปิด `js/config.js` วาง URL:
```js
APPS_SCRIPT_URL: "https://script.google.com/macros/s/XXXXX/exec",
```
เปิด `index.html` → กดปุ่ม ⟳ มุมขวาบนเพื่อดึงรายการล่าสุด
กดปุ่ม 🔒 **เข้าสู่ระบบ** → ใส่รหัส: เจ้าหน้าที่เห็นแอพ/เอกสารที่ซ่อน · ผู้ดูแลได้เมนู 🛠 **จัดการ** + ⚙️ **ตั้งค่า**

## หมายเหตุ
- ถ้า `APPS_SCRIPT_URL` ว่าง หรือออฟไลน์ → Hub ใช้ `APPS_SEED` / แคชล่าสุดแทน (เปิดได้เสมอ)
- แก้ชีทแล้วรายการอัปเดตทันทีที่กด ⟳ (ไม่ต้อง re-deploy)
- เพิ่มแถวจากมือถือผ่านแอป Google Sheets ได้
- เปลี่ยนโค้ด `Code.gs` ต้อง **Deploy ▸ Manage deployments ▸ Edit ▸ New version** ทุกครั้ง
